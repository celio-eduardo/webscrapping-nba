## Web Scraping NBA

### 📌 Descrição

Projeto de extração automatizada de dados estatísticos da NBA, utilizando técnicas de web scraping para coleta, processamento e estruturação de informações públicas disponíveis na web.

O objetivo é demonstrar técnicas de coleta automatizada, tratamento de dados e organização para análise posterior.

---

### ⚙️ Tecnologias Utilizadas

* Python
* Requests
* BeautifulSoup
* Selenium
* CSV

---

### 🧠 Funcionalidades

* Extração automatizada de dados de jogadores/times/estatísticas
* Tratamento e limpeza de dados HTML
* Estruturação em formato tabular
* Exportação para arquivo CSV
* Automação para páginas com carregamento dinâmico

---

### 🚀 Como Executar

1. Clone o repositório:

```
git clone https://github.com/celio-eduardo/webscrapping-nba
```

2. Instale as dependências:

```
pip install -r requirements.txt
```

3. Execute:

```
python programa_principal.py
```
